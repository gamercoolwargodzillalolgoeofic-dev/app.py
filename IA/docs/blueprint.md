# **App Name**: CodeSage AI

## Core Features:

- Intelligent Code Generation: The AI tool understands various programming prompts (short or long) and generates relevant code snippets, functions, or architectural suggestions in the specified or implied language.
- User-Friendly Prompt Editor: An intuitive text input area where users can type or paste their programming prompts with ease, accommodating varying lengths.
- Syntax-Highlighted Code Display: Presents the generated code in a clear, readable format with syntax highlighting suitable for different programming languages.
- Copy-to-Clipboard Functionality: Allows users to easily copy the generated code snippets to their clipboard with a single click for immediate use.

## Style Guidelines:

- The overall theme adopts a dark color scheme to reduce eye strain, commonly preferred in programming environments. The background color is a deep, desaturated blue-gray (#21262B), creating a calm and focused atmosphere.
- The primary color is a vibrant yet refined medium blue (#4799FF), chosen to symbolize intelligence, precision, and technological innovation. It stands out clearly against the dark background.
- An accent color of bright cyan (#2FE2F0) provides energetic contrast and is used for interactive elements like buttons, highlights, and subtle call-to-actions, evoking a modern, digital aesthetic.
- For all general user interface text, headlines, and body content, the 'Inter' sans-serif font is recommended. Its clean, objective, and modern appearance ensures excellent readability and a professional feel. For all generated code snippets, 'Source Code Pro' (monospace sans-serif) is chosen for its clarity and optimal presentation of programming syntax.
- Use minimalist, outline-style icons with a technological and intellectual theme. Icons should reflect common programming concepts (e.g., </> for code, a brain for AI, a copy icon, etc.).
- The layout is clean and highly structured, prioritizing content and functionality. A prominent, focused input area for the prompt is positioned next to or above a clear, expansive display for the generated code, ensuring an efficient workflow. Use ample negative space to maintain visual calm.
- Subtle, fluid animations provide immediate visual feedback. Examples include smooth transitions when new code is generated, subtle glow effects on interactive elements upon hover, and a gentle loading indicator while the AI processes the prompt.
